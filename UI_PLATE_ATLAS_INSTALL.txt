Путь для полного автомата:

`Assets/Resources/UI/ui_plate_atlas.png`

Что будет дальше:
1. Ты кладёшь один atlas PNG по этому пути.
2. Unity сам нарезает его на нужные sprite-имена.
3. Меню и игровые панели сами подхватывают эти спрайты без ручной привязки в инспекторе.

Какие имена режутся автоматически:
- `title_plate`
- `light_panel_plate`
- `button_frame_plate`
- `input_plate`
- `button_text_plate`
- `profile_text_plate`
- `collection_text_plate`
- `lobby_profile_plate`
- `connection_text_plate`
- `participant_panel_plate`
- `participant_text_plate`
- `profile_info_plate`

Если atlas лежит в другом месте:
1. Выдели PNG в `Project`
2. Нажми `Parliament -> UI -> Configure Selected Plate Atlas`

Важно:
- для полного автомата имя файла должно быть `ui_plate_atlas.png`
- для полного автомата путь должен быть `Assets/Resources/UI/`
